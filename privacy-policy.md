Privacy Policy

Last Updated: May 13, 2026

This Privacy Policy explains how the bot ("the Bot") collects, uses, stores, and protects information when users interact with the Bot on Discord.

1. Information We Collect

The Bot may collect and temporarily process the following information:

Discord user IDs

Discord server IDs

Channel IDs

Message IDs

Moderation logs and actions

Settings configured through commands or dashboard features

Temporary message content when required for moderation or automod features

The Bot does not intentionally collect sensitive personal information.

2. How Information Is Used

Information collected is used strictly for:

Providing moderation features

Logging moderation actions

Automod functionality

Saving server configuration settings

Preventing abuse and maintaining service security

Improving bot performance and reliability

3. Data Storage

Some data may be stored in a database to allow the Bot to function correctly, including:

Server configuration

Moderation history

Warning records

Automod rules

Logging settings

Data is retained only as long as necessary for the Bot’s functionality.

4. Third-Party Services

The Bot may use third-party services including:

Discord API

Hosting providers

Database providers

These services may process data according to their own privacy policies.

5. Data Sharing

The Bot does not sell, trade, or rent user data.

Data is only shared when legally required or necessary to prevent abuse, security threats, or violations of applicable laws.

6. User Rights

Server administrators may request removal of server-specific stored data by contacting the Bot owner.

Users may also stop data collection by removing the Bot from a server.

7. Security

Reasonable measures are taken to protect stored information from unauthorized access, misuse, or disclosure.

However, no system can guarantee absolute security.

8. Children’s Privacy

The Bot is not directed toward children under the minimum age required by Discord’s Terms of Service.

9. Changes to This Policy

This Privacy Policy may be updated at any time.

Continued use of the Bot after updates constitutes acceptance of the revised policy.

10. Contact

For questions regarding this Privacy Policy, contact the Bot owner through Discord or the official support server.
