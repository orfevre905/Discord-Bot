Privacy Policy

Last updated: May 13, 2026

Overview

This Privacy Policy explains how this Discord bot (“the Bot”) handles data. We care about transparency, security, and minimal data collection.

What We Collect

The Bot may process:

  • Discord User IDs
  • Server & Channel IDs
  • Message IDs (for moderation features)
  • Moderation logs (warnings, bans, mutes)
  • Server configuration settings

How We Use Data

Data is used strictly for:

  • Moderation tools (warn, ban, mute, automod)
  • Logging actions for server staff
  • Saving server settings
  • Preventing abuse and spam

Storage

Data may be stored in secure databases. We only keep data as long as needed for functionality.

Third Parties

We may use:

  • Discord API
  • Database providers
  • Hosting services

Your Control

Server owners can request deletion of data by removing the bot or contacting support.

Contact

Questions? Contact the bot developer via Discord or support server.
